# Case Técnico — Site Papel de Parede

## Problema

Negócios locais de decoração, reformas e papéis de parede precisam apresentar produtos de forma visual, gerar confiança rapidamente e levar o cliente para um canal de atendimento direto, como WhatsApp. Muitas lojas pequenas dependem apenas de redes sociais, sem uma página própria bem estruturada.

## Solução

O projeto Site Papel de Parede é uma landing page comercial em React para apresentar produtos, estilos, benefícios, oferta e chamada para atendimento via WhatsApp.

## Público-alvo

- Lojas de papéis de parede.
- Lojas de decoração.
- Profissionais de reforma.
- Pequenos e-commerces locais.
- Prestadores de serviço que vendem por WhatsApp.

## Principais funcionalidades

- Hero visual com chamada comercial.
- Catálogo de produtos.
- Filtros por estilo.
- Cards de ambientes.
- Oferta especial.
- Selo de qualidade.
- Benefícios do produto.
- CTA para WhatsApp.
- Layout responsivo.
- Visual preto e dourado com estética premium.

## Arquitetura

```text
React + Vite
     |
     v
Componentes de interface
     |
     v
Dados locais de produtos
     |
     v
CTA externo para WhatsApp
```

## Decisões técnicas

### React com Vite

A escolha permite desenvolvimento rápido, componentização simples, build leve e facilidade para publicação em serviços como Vercel ou GitHub Pages.

### Catálogo com dados locais

Para uma landing page comercial, manter os produtos em arquivo local reduz complexidade e facilita demonstração sem depender de backend.

### CTA para WhatsApp

Negócios locais no Brasil frequentemente fecham vendas pelo WhatsApp. Por isso, o CTA direto aumenta a chance de conversão.

### Design visual premium

A estética preto e dourado foi usada para transmitir valor, elegância e percepção de produto superior.

## Diferenciais para empregabilidade

- Demonstra front-end moderno com React.
- Mostra preocupação com UX/UI.
- Tem proposta comercial clara.
- Aplica componentização.
- Trabalha responsividade.
- Demonstra foco em conversão.

## Riscos e pontos de melhoria

- Adicionar SEO mais completo.
- Criar formulário de orçamento.
- Adicionar cálculo de rolos por parede.
- Adicionar depoimentos.
- Criar painel simples para gerenciar produtos.
- Adicionar testes visuais ou validações básicas.

## Evolução recomendada

1. Adicionar seção de depoimentos.
2. Criar calculadora de quantidade de rolos.
3. Adicionar formulário de orçamento.
4. Melhorar SEO com metatags e dados estruturados.
5. Criar integração com planilha ou CMS simples.
6. Publicar demo estável e adicionar screenshots ao README.

## Como apresentar em entrevista

Este projeto deve ser apresentado como um case de Front-end e UX/UI para negócio local, destacando componentização, responsividade, conversão via WhatsApp e preocupação com apresentação visual de produto.
