# Checklist de Qualidade — Site Papel de Parede

Este checklist organiza melhorias para transformar a landing page em um case forte de Front-end, UX/UI e conversão para negócios locais.

## 1. Interface e experiência

- [ ] Hero com mensagem clara.
- [ ] CTA principal visível acima da dobra.
- [ ] Catálogo carregando corretamente.
- [ ] Filtros funcionando.
- [ ] Cards de produtos bem alinhados.
- [ ] Oferta especial destacada.
- [ ] WhatsApp abrindo corretamente.
- [ ] Rodapé com contato visível.

## 2. Responsividade

- [ ] Mobile até 360px validado.
- [ ] Tablet validado.
- [ ] Desktop validado.
- [ ] Menu mobile funcionando.
- [ ] Imagens ajustadas sem quebrar layout.
- [ ] Botões com boa área de toque.

## 3. Performance

- [ ] Build do Vite sem erro.
- [ ] Imagens otimizadas.
- [ ] Links externos revisados.
- [ ] Sem erros no console.
- [ ] Página carregando rapidamente.

## 4. SEO e conversão

- [ ] Título da página otimizado.
- [ ] Meta description adicionada.
- [ ] Textos com palavras-chave do nicho.
- [ ] Alt text nas imagens.
- [ ] CTA de orçamento claro.
- [ ] Seção de benefícios objetiva.

## 5. Documentação

- [ ] README atualizado.
- [ ] `CASE_TECNICO.md` criado.
- [ ] `QUALITY_CHECKLIST.md` criado.
- [ ] Prints adicionados.
- [ ] Link de demo publicado.

## 6. Melhorias futuras

- [ ] Criar calculadora de rolos.
- [ ] Adicionar depoimentos.
- [ ] Criar formulário de orçamento.
- [ ] Adicionar painel simples para produtos.
- [ ] Criar integração com planilha ou CMS.

## Como apresentar em entrevista

Este projeto deve ser apresentado como case de Front-end, UX/UI e conversão, mostrando capacidade de criar uma página comercial objetiva para negócio local.
